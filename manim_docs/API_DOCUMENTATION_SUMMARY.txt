12. TYPICAL USAGE PATTERNS
================================================================================

Basic Scene:
```python
from manim import *
class MyScene(Scene):
    def construct(self):
        circle = Circle(color=BLUE)
        self.play(Create(circle))
        self.wait(1)
        self.play(FadeOut(circle))
```

Animation Sequence:
```python
self.play(Create(obj))
self.wait(0.5)
self.play(Transform(obj, obj2))
self.play(FadeOut(obj2), run_time=2)
```

Value Tracking:
```python
tracker = ValueTracker(0)
number = DecimalNumber(tracker.get_value())
number.add_updater(lambda m: m.set_value(tracker.get_value()))
self.play(tracker.animate.set_value(10))
```

3D Animations:
```python
class My3DScene(ThreeDScene):
    def construct(self):
        cube = Cube()
        self.move_camera(phi=75*DEGREES)
        self.play(Create(cube))
```

13. COMMAND LINE USAGE
================================================================================

Basic:
- manim scenes.py MyScene

Output options:
- manim -p scenes.py MyScene (preview)
- manim -s scenes.py MyScene (save last frame)
- manim -a scenes.py MyScene (all scenes)

Quality:
- manim -ql scenes.py MyScene (low)
- manim -qm scenes.py MyScene (medium)
- manim -qh scenes.py MyScene (high)
- manim -qk scenes.py MyScene (4K)

Other:
- manim --renderer=opengl scenes.py MyScene
- manim -i scenes.py MyScene (interactive)
